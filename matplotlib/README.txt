
Cities of the world database "worldcities.txt.gz"
was graciously donated to the public by MaxMind.com.

Download it at:
http://www.maxmind.com/download/worldcities/

Before using it, you need to unzip the file manually because for now numpy is 
having a hard time doing it himself.

The database contains locations by country, city, population, latitude and 
longitude. There are over 3,047,000 records in the database.
